# aiwriter
AIWriter: An AI-Powered Assistant for LibreOffice Writer

AIWriter is an open-source extension for LibreOffice Writer designed to seamlessy integrate artificial intelligence into your document editing and writing workflow. Powered entirely by Ollama, this plugin brings the capabilities of advanced Large Language Models (LLMs) directly into your word processor—keeping your data private, secure, and local.

Key Features:

    Local & Private: No API keys, no cloud subscriptions, and no data leaks. Your data stays on your machine.

    Writing Assistance: Brainstorm ideas, draft paragraphs, or expand on notes on the fly.

    Smart Editing: Refine your prose, fix grammar, adjust tone, or summarize long passages instantly.


How it Works

AIWriter offloads all processing to Ollama, a lightweight framework for running LLMs locally. Because it communicates with a local provider, your documents are never uploaded to third-party servers, ensuring 100% data privacy.

Here are the step-by-step instructions to install and configure the **OllamaWriter.oxt** extension in LibreOffice Writer (or Apache OpenOffice). 

### ⚠️ Prerequisites
Before installing, ensure that **Ollama is installed and running** in the background on your computer, and that you have at least one model pulled (e.g., `ollama pull llama3.2`).

---

### Phase 1: Install the Extension
1. **Locate the File:** Find the `OllamaWriter.oxt` file generated by the script on your computer.
2. **Open LibreOffice:** Launch **LibreOffice Writer** (or any LibreOffice application).
3. **Open the Extension Manager:**
   * In the top menu bar, click on **Tools**.
   * Select **Extension Manager...** from the dropdown menu.
4. **Add the Extension:**
   * In the Extension Manager window, click the **Add...** button.
   * A file browser will open. Navigate to where you saved `OllamaWriter.oxt`, select it, and click **Open**.
5. **Accept Prompts:**
   * If a license agreement pops up, read it and click **Accept** (it is an MIT License).
   * If you see a warning about installing an unsigned extension, click **Yes** or **Continue** to proceed.
6. **Verify Installation:** Ensure the extension now appears in the list and its status is set to **Enabled**.
7. **Restart LibreOffice:** Close LibreOffice completely and reopen it. *This is required for the UI menus and toolbars to register.*

---

### Phase 2: Configure the Plugin
Once restarted, the extension needs to know how to talk to your local Ollama server.

1. In the top menu bar, click on the newly added **AI Assistant** menu.
2. Click on **Ollama Configuration Panel...** at the bottom of the dropdown.
3. In the settings window that appears:
   * **Ollama Host URL:** Leave as `http://localhost:11434` (unless you are running Ollama on a different machine/port).
   * **Active Model:** Click the **Fetch** button to pull a list of your installed models from Ollama, then select your preferred model (e.g., `llama3.2`, `mistral`, etc.).
   * **Output Layout:** Choose how you want the AI to format text (Formatted Text is recommended to automatically apply Bold/Headers).
4. Click **Save** and close the window.

---

### Phase 3: Enable the UI Elements (If hidden)
Depending on your LibreOffice setup, the Toolbar or Sidebar might be hidden by default. Here is how to make sure you have full access to the tools:

* **To show the Sidebar Panel:**
  Go to **View** > **Sidebar** (or press `Ctrl+F5`). Look for the **Ollama Assistant** deck/icon on the right side of your screen.
* **To show the Toolbar:**
  Go to **View** > **Toolbars** > **Ollama AI Production Tools**.

---

### Phase 4: Test the Extension
1. Type a few sentences into your LibreOffice Writer document.
2. Highlight the text with your mouse.
3. Test a feature using either the Menu, the Toolbar, or the Keyboard Shortcuts:
   * **Rewrite & Polish:** Press `Ctrl + Shift + R`
   * **Fix Grammar:** Press `Ctrl + Shift + G`
   * **Generate Outline:** Press `Ctrl + Shift + O`
   * **Custom Prompt:** Press `Ctrl + Shift + U`
4. The AI will process your text and replace/insert the result directly into your document!

---

### 🛠️ Troubleshooting
* **Connection Errors:** If you get an "Ollama Connection Error", ensure the Ollama app is actually running in your system tray/taskbar. If you are on Windows, you may need to set the environment variable `OLLAMA_HOST=0.0.0.0` in Windows System Properties if the plugin cannot reach localhost.
* **Extension Not Showing Up:** Go back to **Tools > Extension Manager** and ensure the checkbox next to *Ollama AI Writer Assistant Pro* is checked.
* **Debugging:** If the plugin fails silently, check the debug log file generated on your system. On Windows, it is located at `C:\Users\<YourUsername>\ollama_plugin_debug.log`. On Linux/Mac, it is at `~/ollama_plugin_debug.log`.
